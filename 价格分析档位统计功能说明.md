# 价格分析档位统计功能说明

## 🎯 功能概述

价格分析页面现在按净利率档位统计SKU数量，并支持点击跳转到SKU详情页面进行筛选查看。

## 📊 净利率档位分类

### 💚 健康档位 (≥20%)
- **标准**: 净利率 ≥ 20%
- **含义**: 盈利能力强，商品表现优秀
- **颜色**: 绿色 (#7ED321)

### 💛 一般档位 (10-20%)
- **标准**: 10% ≤ 净利率 < 20%
- **含义**: 盈利能力中等，商品表现正常
- **颜色**: 黄色 (#F5A623)

### 🟠 注意档位 (0-10%)
- **标准**: 0% ≤ 净利率 < 10%
- **含义**: 盈利微薄，需要关注优化
- **颜色**: 橙色 (#FF9500)

### 🔴 亏损档位 (<0%)
- **标准**: 净利率 < 0%
- **含义**: 商品亏损，需要紧急处理
- **颜色**: 红色 (#D0021B)

## 🖱️ 交互功能

### 点击跳转
- 点击任意档位的统计卡片
- 自动跳转到SKU列表页面
- 应用对应的净利率筛选条件
- 只显示符合该档位条件的商品

### 筛选逻辑
- **健康档位**: 只显示净利率 ≥ 20% 的商品
- **一般档位**: 只显示净利率在 10-20% 之间的商品
- **注意档位**: 只显示净利率在 0-10% 之间的商品
- **亏损档位**: 只显示净利率 < 0% 的商品

## 💡 使用场景

### 商品优化
1. 点击 🔴 亏损档位，查看需要紧急处理的商品
2. 分析亏损原因：采购价过高、定价过低等
3. 制定优化策略：调整价格、更换供应商等

### 利润分析
1. 点击 💚 健康档位，学习优秀商品的特点
2. 点击 🟠 注意档位，寻找提升空间
3. 对比不同档位商品的共同特征

### 库存管理
1. 优先推广健康档位商品
2. 谨慎进货注意档位商品
3. 考虑清仓亏损档位商品

## 🔢 计算公式

### 净利率计算
```
净利率 = (毛利率 - 杂费率) × 100%
毛利率 = (到手价 - 采购价 - 快递费) / 到手价
杂费率 = 10% (售后2% + 管理7% + 平台1%)
```

### 快递费规则
- 到手价 ≥ 150元：快递费 = 30元
- 到手价 < 150元：快递费 = 2元
